#  Eaye蛇 — 经典贪吃蛇多模式挑战

一款功能丰富的网页贪吃蛇游戏，支持 **8种游戏模式**、**16套主题**、**7种食物**、**双人对战**、**关于页面** 等特性。最新版本 v3.0。

> 🌐 在线体验：https://vxxy77.github.io/eaye-snake/

---

## 🎮 游戏模式

| 模式 | 说明 |
|------|------|
| 🏆 **闯关** | 8关递增挑战，每关目标分递增，速度逐关加快，通关🎉庆祝 |
| 🤖 **AI** | 双蛇同场竞技，AI使用贪心算法自动寻路，比拼得分 |
| 🧱 **迷宫** | 固定墙壁障碍（4种模板轮流），碰墙即死 |
| ⚔️ **对战** | 支持 1P vs AI 或 2P 双人同屏对战 |
| ⏱ **限时** | 60秒倒计时冲刺，最后10秒画面闪红 |
| ♾ **无尽** | 不会自撞死亡，蛇可穿过自身，无限畅玩 |
| 🎯 **道具赛** | 随机生成道具：🛡️护盾 ⚡加速 🐢减速 💎+5分 |
| 📅 **每日** | 每日固定种子迷宫+限时挑战，最高分本地记录 |

## 👥 双人游玩

在对战模式下，点击菜单底部 `2P` 按钮启用双人模式：

| 玩家 | 控制键 | 蛇身颜色 |
|------|--------|----------|
| 🔵 P1 | `↑ ↓ ← →` | 跟随主题渐变色 |
| 🔴 P2 | `W A S D` | 红橙渐变色 |

## ⌨️ 操作说明

| 按键 | 功能 |
|------|------|
| `↑ ↓ ← →` | P1 方向控制 |
| `W A S D` | P2 方向控制（双人模式） |
| `空格` | 开始游戏 / 暂停/继续 |
| `ESC` | 返回菜单 |
| 触屏滑动手势 / 方向键 | 移动端方向控制 |

## 📊 游戏统计

游戏自动记录：累计总局数、吃食物总数、最长蛇身记录，在「关于」页面查看。

## 🔊 音效

使用 Web Audio API 生成简单音效：吃食物、死亡、过关、按钮点击均有反馈。
