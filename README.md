# 🐍 Eaye蛇 — 经典贪吃蛇多模式挑战

一款功能丰富的网页贪吃蛇游戏，支持 **8种游戏模式**、**16套主题**、**7种食物**、**双人对战**、**连击系统**、**难度选择**、**蛇皮换肤**、**平滑动画** 等特性。最新版本 **v3.40**。

> 🌐 在线体验：https://vxxy77.github.io/eaye-snake/

---

## 🎮 游戏模式

| 模式 | 说明 |
|------|------|
| 🏆 **闯关** | 8关递增挑战，每关随机墙体，速度逐关加快 |
| 🤖 **AI** | 双蛇同场竞技，AI贪心算法自动寻路 |
| 🧱 **迷宫** | 固定墙壁障碍（4种模板轮流），碰墙即死 |
| ⚔️ **对战** | 支持 1P vs AI 或 2P 双人同屏对战 |
| ⏱ **限时** | 60秒倒计时冲刺，最后10秒画面闪红 |
| ♾ **无尽** | 不撞自身 + 动态障碍（每15食物生成一个障碍） |
| 🎯 **道具赛** | 随机生成道具：🛡️护盾 ⚡加速 🐢减速 💎+5分 |
| 📅 **每日** | 每日固定种子迷宫 + 限时挑战 |

## 🍎 食物系统

- 每 **2秒** 自动生成1个食物，最多 **6个** 同屏
- 每种食物显示分值

| 食物 | 分值 | 说明 |
|------|------|------|
| 🍎 苹果 | +1 | 最常见 |
| 🍊 橘子 | +2 | 常见 |
| 🍇 葡萄 | +3 | 中等 |
| 🍒 樱桃 | +4 | 较少 |
| 🍉 西瓜 | +5 | 稀有 |
| 🍀 四叶草 | 道具 | 激活🔥双倍积分（约5秒） |
| ⭐ 星星 | +8 | 最稀有 |

## 🔥 连击系统

- 5秒内连续吃食物触发连击
- **2连击×2倍** → **3连击×3倍** → **4连击×4倍** → **5连击×5倍**（最高）
- 屏幕底部显示连击状态条 + 飘字提示

## 👥 双人游玩

| 玩家 | 控制键 | 蛇身颜色 |
|------|--------|----------|
| 🔵 P1 | `↑ ↓ ← →` | 跟随主题渐变色 |
| 🔴 P2 | `W A S D` | 红橙渐变色 |

## ⚡ 难度选择

| 难度 | 速度 | 分数倍率 |
|------|------|---------|
| 🟢 休闲 | 200ms | ×0.5 |
| 🟢 简单 | 170ms | ×0.7 |
| 🔵 普通 | 140ms | ×1.0 |

## 🎨 主题系统

内置 **16 套视觉主题**，每套主题有独特纹理和渐变：

经典蓝 · 暗夜黑 · 森林绿 · 日落橙 · 极光紫 · 海洋青 · 蜜糖琥珀 · 樱花粉 · 薄荷青 · 熔岩红 · 星空蓝 · 暖阳黄 · 石墨灰 · 霓虹绿 · 葡萄紫 · 🌈彩虹

## 🐍 蛇皮选择

独立于主题的蛇身皮肤，5种可选：

| 皮肤 | 效果 |
|------|------|
| 渐变 | 从蛇头到蛇尾由深到浅（默认） |
| 纯色 | 通体使用主题色 |
| 条纹 | 深浅交替的斑马纹 |
| 彩虹 | HSL 彩虹色流动 |
| 霓虹 | 霓虹高亮色流动 |

## ✨ 平滑动画

蛇身每节在移动时进行 **60fps 插值过渡**，告别瞬移跳跃感。

## 📏 网格自定义

关 / 淡 / 中 / 深 四级可调，游戏中随时切换。

## 📊 生涯数据

自动记录：累计总局数、总得分、平均每局得分、吃食物总数、最长蛇身、最长存活时间。在「关于」页面查看。

## 🔊 音效

右上角 🔊/🔇 按钮开关。包含：吃食物、死亡、过关、按钮点击、道具拾取、倒计时GO音效。

## ⏸ 自动暂停

切换标签页时游戏自动暂停，切回来手动继续。

## ⌨️ 操作说明

| 按键 | 功能 |
|------|------|
| `↑ ↓ ← →` | P1 方向控制 |
| `W A S D` | P2 方向控制（双人模式） |
| `空格` | 开始游戏 / 暂停/继续 |
| `ESC` | 返回菜单 |
| `?` 或 `/` | 显示全部快捷键帮助 |
| 触屏滑动手势 | 移动端方向控制 |

## 💀 游戏结束详情

游戏结束时显示本局数据：蛇身节数、存活秒数、吃食物数量。
